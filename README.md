# Iyang's Place - Where Every Girl's Story Matters

A safe, empowering online community where young women share real stories, find inspiration, and connect through honest girl talk.

## About

Iyang's Place is a digital sisterhood that provides a safe space for young women to:
- Share authentic personal stories
- Find inspiration and support
- Connect through real conversations
- Explore topics like faith, relationships, career, and personal growth

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd sisters-story-share-main

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## Technologies Used

- **Vite** - Build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn/ui** - Component library
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **React Hook Form** - Form handling
- **Zod** - Schema validation
